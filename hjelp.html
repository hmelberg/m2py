Microdata Script Runner

Skriv og kjør microdata.no-kode, Python og R direkte i nettleseren — uten installasjon.

Hva er dette?

Script Runner er et verktøy for å skrive og kjøre kode mot norske registerdata fra microdata.no. Du kan skrive microdata-kommandoer og se resultater umiddelbart, oversette mellom Python/R og microdata-syntaks, og kjøre hybridskript der ulike språk samarbeider.

NB: Dette er et hobbyprosjekt og er ikke laget av microdata.no selv. Siden inneholder ikke ekte tall, og det gis ingen garantier for at analysene er korrekt implementert.

Alt skjer i nettleseren — ingen data forlater maskinen din.

Hurtigstart: Skriv microdata-kommandoer i tekstfeltet, trykk Ctrl+Enter eller knappen Kjør. Bruk Tab for forslag til kommandoer og variabler.

Kjøre og oversette

Kjøre direkte

Trykk Ctrl+Enter eller knappen Kjør. Resultater vises i outputfeltet. Tabeller, diagrammer og tekst vises automatisk.

Oversette

Knappen Oversett (pil-ikon) konverterer Python- eller R-kode til microdata-syntaks. Resultatet vises i outputfeltet — for å kjøre den oversatte koden må du enten kopiere den inn i editoren og bytte til microdata-modus, eller skrive den om til microdata direkte.

Stoppe / avbryte

Trykk Kjør-knappen igjen under kjøring for å avbryte.

Stoppunkter

Klikk på et linjenummer for å sette et rød stoppunkt. Under Kjør skrittvis vil kjøringen pause ved stoppunktet og vente på at du trykker videre.

Tab-autocomplete

Trykk Tab i editoren for kontekstbevisste forslag: kommandoer, variabelnavn, DataFrame-metoder og mer. Etter / (f.eks. import db/) foreslås microdata-variabler fra katalogen. På tom linje foreslås kommandoer.

Tips: Trykk Tab midt i et ord for å filtrere listen etter det du allerede har skrevet. Se fullstendig referanse for alle kontekstregler.

Datasett-panel (høyre sidemeny)

Sidemenyen til høyre viser alle datasett som er lastet inn i gjeldende kjøring.

  • Antall variabler og rader per datasett
  • Variabelliste med type-ikon: ABC tekst, 123 numerisk, skjold-ikon for pseudonymisert ID
  • Klikk på en variabel for detaljvisning: navn, type, beskrivelse og lenke til microdata-katalogen

Sidemenyen oppdateres automatisk etter hver kjøring og etter hvert blokk i Kjør skrittvis-modus.

Kjør skrittvis — steg-for-steg gjennomgang

Kjør skrittvis-knappen kjører scriptet ett blokk om gangen og leser opp kommentarene med tale. Alle kommentarlinjer som starter med // (microdata) eller # (Python/R) blir lest høyt — kode-linjene utføres uten kommentar. Nyttig for presentasjoner og undervisning.

Blokker

En blokk er en gruppe sammenhengende linjer uten tomlinjer imellom. Én eller flere tomlinjer separerer blokker.

// Denne kommentaren leses OPP før koden kjøres
create-dataset df
import db/BEFOLKNING_KJOENN as kjonn

// Dette er neste blokk
summarize kjonn
// Denne kommentaren leses etter at resultatet vises

Kommentarsyntaks

SyntaksModusEffekt
// tekstMicrodataLeses som tale; foranstilt kommentar leses FØR koden, etterhengt leses ETTER
# tekstPython RSamme som ovenfor
//- eller #-AlleStille kommentar — første kodeord leses ikke høyt, og tekst etter - leses heller ikke
//(lang=nb-NO,rate=1.2) tekstAlleTale med tilpassede parametere (språk, hastighet, stemme, toneleie)

Pause, navigasjon og stoppunkter

  • Trykk Kjør skrittvis igjen for å pause/fortsette
  • Pil-knappene hopper til forrige/neste blokk
  • Stoppunkter (klikk på linjenummer) gjør at kjøringen pauser automatisk

//widget NAVN { ...json... }

Blokk-syntaks for lengre innhold

For HTML, SVG og markdown med flere linjer kan du bruke blokk-syntaks i stedet for JSON:

//start_widget:html(title="Eksempel", speak="Her ser du HTML-innhold.")
<div style="text-align:center">
  <h2>Hei fra HTML</h2>
  <p>Flere linjer uten escaping.</p>
</div>
//end_widget

Moduser

Velg modus fra dropdown-menyen i bunnbaren, eller bruk modusveksler-linjer i scriptet for å veksle underveis (se Hybrid-skript).

Microdata.no-syntaks. Laster registerdata, oppretter datasett, kjører statistikk og diagrammer.
Python Pyodide

  • Matplotlib, Seaborn, Plotly
  • Scikit-learn, statsmodels, lifelines

Oversettelse til microdata

Følgende Python-mønstre oversettes godt av py2m:

  • df = pd.read_csv(...) → import-kommandoer
  • df.groupby("x")["y"].mean() → collapse (mean) y, by(x)
  • df[df["x"] > 0] → keep if x > 0
  • df["ny"] = df["x"] + df["y"] → generate ny = x + y
  • df.rename(columns={"a": "b"}) → rename a b
  • df.merge(df2, on="id") → merge

Kode som bruker egendefinerte klasser, list comprehensions, rekursjon, asynkron logikk eller komplekse lambda-uttrykk oversettes ikke og må skrives om.

R (WebR)

R kjøres via WebR (WebAssembly). Pakker installeres automatisk første gang de brukes (library(dplyr) osv.).

Oversettelse til microdata

Følgende R-mønstre oversettes godt av r2m:

  • df %>% filter(x > 0) → keep if x > 0
  • df %>% select(x, y) → keep x y
  • df %>% mutate(ny = x + y) → generate ny = x + y
  • df %>% group_by(g) %>% summarise(m = mean(x)) → collapse (mean) x -> m, by(g)
  • df %>% rename(ny = gammel) → rename gammel ny
  • left_join(df, df2, by = "id") → merge

Kode som bruker purrr, egendefinerte R6-klasser, Shiny, miljømanipulasjon eller komplekse NSE-uttrykk oversettes ikke automatisk.

Hybrid-skript

Du kan blande microdata, Python og R i samme script ved å bruke modusveksler-linjer.

Vekslerlinjer

LinjeEffekt
## microdata / #micro / // microdataBytt til microdata-modus
## python / #pyBytt til Python-modus
## r / #rBytt til R-modus

Eksempel: microdata + Python

## microdata
require no.ssb.fdb:52 as db
create-dataset df
import db/BEFOLKNING_FOEDSELS_AAR_MND as faarmnd
import db/INNTEKT_WLONN 2022-01-01 as inntekt
generate alder = 2022 - int(faarmnd / 100)
keep if inrange(alder, 25, 67)
keep if inntekt > 0

## python
import matplotlib.pyplot as plt
df.plot.scatter(x="alder", y="inntekt", alpha=0.3, s=8)
plt.tight_layout()
show(plt.gcf())

Eksempel: microdata + R

# microdata
require no.ssb.fdb:52 as db
create-dataset df
import db/BEFOLKNING_KJOENN as kjonn
import db/INNTEKT_WLONN 2022-01-01 as inntekt

#r
library(dplyr)
df %>% group_by(kjonn) %>%
  summarise(snitt = mean(inntekt, na.rm = TRUE))

Datasettene fra microdata-seksjonen overføres automatisk til Python som pandas DataFrames og til R som data frames.
